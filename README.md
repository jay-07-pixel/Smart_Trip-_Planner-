## 🚀 **Getting Started**

### **Prerequisites**
- Modern web browser (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+)
- Internet connection for API access
- **API Keys** (for full functionality):
  - Google Places API key
  - OpenWeatherMap API key

### **Quick Start (No Setup Required)**
Simply visit our **[Live Demo](https://ai-trip-planner-india.netlify.app)** to start planning immediately!

### **Local Development Setup**

#### **Method 1: Direct Browser (Fastest)**
1. **Download the project**
   ```bash
   git clone https://github.com/jay-07-pixel/Smart_Trip-_Planner-.git
   cd Smart_Trip-_Planner-
   ```

2. **Open index.html** in your browser
   - Double-click `index.html` file
   - Or drag and drop it into your browser

#### **Method 2: Local Server (Recommended)**
1. **Clone and navigate**
   ```bash
   git clone https://github.com/jay-07-pixel/Smart_Trip-_Planner-.git
   cd Smart_Trip-_Planner-
   ```

2. **Start local server**
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Python 2
   python -m SimpleHTTPServer 8000
   
   # Node.js (if you have npx)
   npx http-server -p 8000
   
   # PHP (if installed)
   php -S localhost:8000
   ```

3. **Access the application**
   - Open your browser
   - Visit `http://localhost:8000`
   - Start planning your trip!

#### **Method 3: Live Server (VS Code)**
1. Install **Live Server** extension in VS Code
2. Right-click on `index.html`
3. Select **"Open with Live Server"**

## 🔧 **Configuration & API Setup**

### **🔑 API Keys Setup (Required for Full Functionality)**

#### **Step 1: Google Places API Key**
1. **Go to Google Cloud Console**
   - Visit [Google Cloud Console](https://console.cloud.google.com/)
   - Create a new project or select existing one

2. **Enable APIs**
   - Navigate to "APIs & Services" → "Library"
   - Enable these APIs:
     - **Places API**
     - **Maps JavaScript API**
     - **Distance Matrix API**

3. **Create API Key**
   - Go to "APIs & Services" → "Credentials"
   - Click "Create Credentials" → "API Key"
   - Copy your API key

4. **Configure API Key**
   ```javascript
   // In your JavaScript files, replace:
   const GOOGLE_API_KEY = 'YOUR_GOOGLE_API_KEY_HERE';
   ```

#### **Step 2: OpenWeatherMap API Key**
1. **Sign up at OpenWeatherMap**
   - Visit [OpenWeatherMap](https://openweathermap.org/api)
   - Create a free account

2. **Get your API key**
   - Go to your profile → "My API keys"
   - Copy the default API key (or create a new one)

3. **Configure API Key**
   ```javascript
   // In your JavaScript files, replace:
   const WEATHER_API_KEY = 'YOUR_WEATHER_API_KEY_HERE';
   ```

#### **Step 3: Create Local Config File**
1. **Copy the example config**
   ```bash
   cp js/config.example.js js/config.js
   ```
   On Windows (PowerShell):
   ```powershell
   Copy-Item js\config.example.js js\config.js
   ```

2. **Add your API keys** in `js/config.js`:
   ```javascript
   window.API_CONFIG = {
       GOOGLE_API_KEY: 'YOUR_GOOGLE_API_KEY_HERE',
       WEATHER_API_KEY: 'YOUR_OPENWEATHER_API_KEY_HERE'
   };
   ```

> **Important:** `js/config.js` is listed in `.gitignore` and must never be committed. Only `js/config.example.js` (with placeholders) is tracked in Git.

### **🔒 Security Best Practices**
- **Never commit API keys** to public repositories
- Use **environment variables** for production
- Set up **API key restrictions** in Google Cloud Console
- Monitor **API usage** to avoid unexpected charges
